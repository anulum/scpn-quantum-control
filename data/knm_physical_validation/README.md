# K_nm Physical Validation Data

This directory stores small derived measured-coupling artifacts for the
K_nm physical-validation audit. Raw EEG files are not committed; they are
downloaded locally under `.coordination/datasets/` and identified by SHA-256.

`measured_couplings.json` is generated by:

```bash
.venv-linux/bin/python scripts/build_real_eeg_plv_validation_dataset.py --download
```

The current artifact uses PhysioNet EEGMMIDB subject 1 run 1
(`S001R01.edf`, baseline eyes open), extracts the canonical 8-channel subset,
computes 8-13 Hz alpha-band phase-locking values, and estimates uncertainty as
the standard error across overlapping PLV windows.
