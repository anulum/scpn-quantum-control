# K_nm Physical Validation Data

This directory stores small derived measured-coupling artifacts for the
K_nm physical-validation audit. Raw EEG files are not committed; they are
downloaded locally under `.coordination/datasets/` and identified by SHA-256.

Artifacts:

| Artifact | Cohort | Contents |
|---|---|---|
| `measured_couplings.json` | Baseline eyes open, `S001R01` through `S109R01`. | 28 cohort-median alpha-band PLV edges plus 3,052 per-record edge estimates. |
| `measured_couplings_baseline_closed.json` | Baseline eyes closed, `S001R02` through `S109R02`. | 28 cohort-median alpha-band PLV edges plus 3,052 per-record edge estimates. |
| `baseline_open_closed_comparison.json` | Eyes closed minus eyes open. | Per-edge condition deltas and descriptive summary statistics. |
| `eeg_alpha_plv_knm_comparison.json` | Baseline eyes open versus eight-layer K_nm. | Topology, magnitude, spectral, null-model, and promotion-readiness diagnostics for the primary EEG PLV candidate. |
| `measured_couplings_power_grid_ieee5bus.json` | IEEE 5-bus public benchmark constants. | 10 swing-equation coupling edges with raw susceptance, voltage, inertia, and propagated rounding uncertainty. |
| `power_grid_ieee5bus_knm_comparison.json` | IEEE 5-bus versus five-layer K_nm. | Topology and magnitude diagnostics for the measured-system control. |
| `measured_couplings_power_grid_ieee14bus.json` | IEEE 14-bus public benchmark constants. | 91 voltage-weighted admittance candidate edges with public branch reactance, voltage, and propagated rounding uncertainty. |
| `power_grid_ieee14bus_knm_comparison.json` | IEEE 14-bus versus fourteen-layer K_nm. | Topology, magnitude, spectral, null-model, and measured-system promotion-readiness diagnostics. |

The eyes-open artifact is generated by:

```bash
.venv-linux/bin/python scripts/build_real_eeg_plv_validation_dataset.py \
  --download --record-preset baseline-open-109
```

The eyes-closed artifact is generated by:

```bash
.venv-linux/bin/python scripts/build_real_eeg_plv_validation_dataset.py \
  --download --record-preset baseline-closed-109 \
  --output data/knm_physical_validation/measured_couplings_baseline_closed.json
```

The condition comparison is generated by:

```bash
.venv-linux/bin/python scripts/compare_eeg_plv_cohorts.py
```

The eyes-open EEG PLV versus K_nm audit artifact is generated by:

```bash
.venv-linux/bin/python scripts/run_knm_physical_validation_audit.py \
  --measured data/knm_physical_validation/measured_couplings.json \
  --n-layers 8 \
  --output data/knm_physical_validation/eeg_alpha_plv_knm_comparison.json
```

The IEEE 5-bus power-grid measured-system artifact is generated by:

```bash
.venv-linux/bin/python scripts/build_power_grid_measured_couplings.py
```

The IEEE 14-bus voltage-weighted admittance candidate is generated by:

```bash
.venv-linux/bin/python scripts/build_power_grid_measured_couplings.py \
  --case ieee14
```

Both cohort artifacts extract the canonical 8-channel subset, compute 8-13 Hz
alpha-band phase-locking values, and report the median coupling per edge with
median-absolute-deviation uncertainty across records. Per-record edge estimates
are retained inside the JSON artifacts so future audits can recompute cohort
medians and dispersion without the raw EDF files.

The 2026-04-30 closed-vs-open comparison reports:

- open mean edge PLV: `0.545586`;
- closed mean edge PLV: `0.600050`;
- mean closed-minus-open delta: `0.054463`;
- median closed-minus-open delta: `0.061094`;
- mean absolute edge delta: `0.066249`;
- largest absolute edge delta: `0.135901` on edge `(2, 3)`;
- Pearson correlation across edge medians: `0.963961`.

The EEG PLV versus eight-layer K_nm audit reports `28` matched edges,
Spearman topology correlation `0.767804`, and Pearson topology correlation
`0.715303`. Its measured-system promotion decision remains
`blocked_measured_system_promotion_gate` because `phase_locking_value` is an
association observable rather than a calibrated K_nm coupling magnitude, and
because magnitude and spectral gates remain outside tolerance.

This is a condition-specific EEG PLV comparison. It does not validate K_nm as a
physical-unit measured coupling law; that remains a separate measured-system
promotion gate.

The 2026-04-30 IEEE 5-bus measured-system comparison uses the locked conversion
`K_ij = V_i V_j B_ij / (2 H_i omega_0)` with arithmetic symmetrisation of the
two directed inertia terms. Against the canonical five-layer K_nm matrix, the
audit reports:

- matched edges: `10`;
- Spearman topology correlation: `0.190394`;
- Pearson topology correlation: `0.226144`;
- direct RMSE: `0.219843`;
- direct relative RMSE versus mean absolute measured coupling: `261.335679`;
- best scale through origin: `0.003925`;
- scaled RMSE: `0.000894`;
- scaled relative RMSE versus mean absolute measured coupling: `1.063045`;
- maximum direct absolute error: `0.300747`.

Spectral diagnostics compare the measured five-bus subgraph to the canonical
five-layer K_nm subgraph:

- weighted adjacency spectrum Pearson: `0.826329`;
- weighted adjacency spectrum RMSE: `0.439359`;
- weighted Laplacian spectrum Pearson: `0.829195`;
- weighted Laplacian spectrum RMSE: `0.966608`;
- dimensionless Kuramoto threshold-proxy ratio measured/canonical: `244.239277`.

Null-model diagnostics now test two preregistered controls:

- all `120` node-label permutations, preserving measured graph weights and
  degree sequence;
- `4,096` seeded edge-value permutations, preserving the measured value
  distribution.

The observed IEEE 5-bus comparison does not beat either null family at the
95 % gate. Node-label null empirical p-values are `0.363636` for Spearman,
`0.264463` for Pearson, and `0.264463` for RMSE. Edge-value null empirical
p-values are `0.291189` for Spearman, `0.254577` for Pearson, and `0.254577`
for RMSE.

This is a measured-system negative/control result. It keeps K_nm physical
validation open because the power-grid magnitudes, spectra, and threshold proxy
do not match canonical K_nm within the propagated uncertainty and topology
correlation is weak.

The measured-system promotion gate also checks the coupling-unit class.
Association observables such as phase-locking value, coherence, correlation,
mutual information, and transfer entropy are blocked from promotion as
calibrated K_nm magnitudes even when their preprocessing and normalisation are
locked. Model-derived power-grid coupling units pass that unit-class screen,
but they still cannot promote unless the per-edge uncertainty, direct
magnitude, spectral response, and null-model gates also pass.
