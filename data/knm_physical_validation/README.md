# K_nm Physical Validation Data

This directory stores small derived measured-coupling artifacts for the
K_nm physical-validation audit. Raw EEG files are not committed; they are
downloaded locally under `.coordination/datasets/` and identified by SHA-256.

`measured_couplings.json` is generated by:

```bash
.venv-linux/bin/python scripts/build_real_eeg_plv_validation_dataset.py \
  --download --record-preset baseline-open-109
```

The current artifact uses the 109-subject PhysioNet EEGMMIDB baseline
eyes-open cohort (`S001R01` through `S109R01`), extracts the canonical
8-channel subset, computes 8-13 Hz alpha-band phase-locking values, and reports
the median coupling per edge with median-absolute-deviation uncertainty across
records. Per-record edge estimates are retained inside the JSON artifact so
future audits can recompute the cohort median and dispersion without the raw EDF
files.
