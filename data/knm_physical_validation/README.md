# K_nm Physical Validation Data

This directory stores small derived measured-coupling artifacts for the
K_nm physical-validation audit. Raw EEG files are not committed; they are
downloaded locally under `.coordination/datasets/` and identified by SHA-256.

`measured_couplings.json` is generated by:

```bash
.venv-linux/bin/python scripts/build_real_eeg_plv_validation_dataset.py --download \
  --record S001R01 --record S002R01 --record S003R01 --record S004R01 \
  --record S005R01 --record S006R01 --record S007R01 --record S008R01 \
  --record S009R01 --record S010R01
```

The current artifact uses a 10-record PhysioNet EEGMMIDB baseline eyes-open
cohort (`S001R01` through `S010R01`), extracts the canonical 8-channel subset,
computes 8-13 Hz alpha-band phase-locking values, and reports the median
coupling per edge with median-absolute-deviation uncertainty across records.
