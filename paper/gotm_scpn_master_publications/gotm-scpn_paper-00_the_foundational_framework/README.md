# GOTM-SCPN Paper 0: The Foundational Framework

This package contains the Paper 0 source material, tuned full-fidelity
extraction, source-validation artefacts, scientific synthesis, validation
planning, experiment packages, results, and theory-revision candidates.

## Evidence-Class Layout

- `source/` - canonical source dump and source metadata.
- `extraction/` - extracted equations, definitions, mechanisms, flags, and
  extraction index.
- `source_validation_artifacts/` - stable source ledgers, validation specs,
  fixture outputs, promotion gates, media, and reconciliation reports.
- `synthesis/` - derived scientific synthesis and assessment documents.
- `validation_protocols/` - protocols for testing Paper 0 claims.
- `experiments/` - prepared experiment packages and run plans.
- `results/` - measured-system, simulator, QPU, or provider evidence.
- `revisions/` - proposed corrections back into the foundational text.

Do not mix these evidence classes. In particular, source extraction is not a
validation result, and synthesis is not an experiment.
