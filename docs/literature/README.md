# Literature Catalogue

This directory stores project-relevant literature surveys and citation-planning
material. Keep literature reviews here instead of the repository root.

## Current Catalogue

| File | Scope | Status |
|---|---|---|
| `literature/literature_survey_2024_2026.md` | Quantum computing advances relevant to SCPN Quantum Control, including Kuramoto simulation, mitigation, variational algorithms, QEC, and hardware platforms. | Working literature survey. |

## Placement Rule

- Literature surveys and citation catalogues belong under `docs/literature/`.
- Submission-specific bibliographies belong inside the corresponding
  `paper/submissions/...` or `paper/submissions_joss/...` package.
- Internal-only reviewer notes, outreach notes, and grant-specific literature
  scans belong under `private internal records/`.
