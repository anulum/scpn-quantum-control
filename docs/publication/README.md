# Publication Operations

This directory contains submission checklists, source-packaging notes, registry
plans, and venue-readiness material.

Keep manuscript source packages under `paper/`. Keep cross-paper planning under
`paper/publication_planning/`. This directory is for publication operations that
belong in public project documentation but should not clutter the docs root.
